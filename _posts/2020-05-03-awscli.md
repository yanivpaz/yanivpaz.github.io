##  no need for grep , tr ,awk and jq

#### Before
```
aws s3api list-buckets |jq ".Buckets[].Name"  | grep cf | tr -d '"'
```

#### After
```
aws s3api list-buckets --query 'Buckets[?starts_with(Name, `cf-`) == `true`].[Name]'  --output text
```
